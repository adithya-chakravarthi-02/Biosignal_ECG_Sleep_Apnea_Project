# Figures

This folder contains the figures generated during the evaluation of the ECG-based Sleep Apnea Detection model.

## Figures

- **Confusion_Matrix.png** – Confusion matrix summarizing the classification results.
- **ROC.png** – Receiver Operating Characteristic (ROC) curve with the corresponding Area Under the Curve (AUC).
- **Training_Loss.png** – Training and validation binary cross-entropy loss during model training.

These figures provide a visual summary of the model's performance.
